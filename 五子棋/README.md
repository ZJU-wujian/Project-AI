# 现代五子棋

一个现代风格的五子棋游戏，支持双人对战和人机对战。

## 功能

- 双人对战
- 人机对战（简单AI）
- 胜负计数（持久化存储）
- 先手不能形成两个活三
- 提示按钮（推荐最优落子点）

## 如何运行

1. 打开 `index.html` 在浏览器中。
2. 选择游戏模式（双人对战或人机对战）。
3. 点击“开始游戏”。
4. 点击棋盘落子。
5. 在人机对战中，电脑会自动落子。

## 规则

- 黑子先手。
- 先手不能落子形成两个活三。
- 连成五子获胜。
- 如果棋盘满且无人获胜，则平局。

## 技术

- HTML/CSS/JavaScript
- 使用 localStorage 存储胜负计数
